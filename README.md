# Expats Argentina

A comprehensive SEO-focused website for expats considering a move to Argentina. Sponsored by Lucero Legal.

## Domain

**expatsargentina.com** (available for registration)

## Features

- **Visa Guides** - Digital Nomad, Work, Retirement, Student, Investment visas
- **Cost of Living** - Real budget breakdowns with 2025 data
- **Neighborhood Guides** - Detailed reviews of Palermo, Recoleta, Belgrano, San Telmo, and more
- **Healthcare** - Insurance options, hospitals, medical care
- **Housing** - Renting guide, avoiding scams, contracts
- **Banking** - Money transfers, blue dollar, opening accounts
- **Expat Stories** - Real experiences from people living in Argentina
- **Resources** - Checklists, directories, and tools

## Tech Stack

- Next.js 16 with App Router
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Static export for fast hosting

## Development

```bash
cd my-app
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output is in `dist/` directory, ready for deployment to Vercel, Netlify, or any static host.

## Deployment to Vercel

1. Push to GitHub
2. Connect to Vercel
3. Set build command: `npm run build`
4. Set output directory: `dist`
5. Add domain: `expatsargentina.com`
